# ☄️ Binary Logistic Regression for Predicting Hazardous Asteroids

## 📌 Project Overview

This project uses binary logistic regression to predict whether an asteroid is potentially hazardous using NASA data.

## 📊 Dataset

* Source: NASA asteroid dataset
* Target variable: is_potentially_hazardous
* Observations: 64,000+

## 🧠 Methods

* Logistic Regression (GLM)
* Data preprocessing
* Train-test split (80/20)
* Threshold comparison (0.5 vs 0.2)
* ROC Curve & AUC

## 📈 Results

* AUC ≈ 0.94
* Strong classification performance
* Improved sensitivity with threshold adjustment

## 🔍 Key Insights

* Hazardous asteroids tend to have higher velocity and larger diameter
* Accuracy alone is misleading in imbalanced datasets
